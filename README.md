# Java + Spring Boot Mentorship — Code

This is my personal code repo for a 12-week Java + Spring Boot mentorship with my mentor, Titus Muthomi.

The curriculum, milestones, and DSA problem list live in a separate playbook repo. This repo is where all the actual code goes.

## What's here

- **`exercises/`** — Weeks 1–6 learning exercises (Java fundamentals → Spring Boot basics), organized by week
- **`capstone/`** — The main project, picked in week 7 and built out through week 12

## Working agreements

- Daily commits to a feature branch, PRs opened into `main`
- Every PR explains what it does and how to test it
- Mentor reviews within ~24 hours; feedback addressed before merging
